## About Wave

Wave is a Roblox script executor made for Windows users who want a compact desktop app for Lua injection, script execution, and access to a built-in script hub. It pairs a straightforward WPF interface with practical functions like batch execution, a persistent queue, and an auto-update engine so it remains useful as builds change over time.

The experience is built around convenience: no key required, an integrated debugger, multilingual UI support, and a bundled library of 500+ scripts. If you are looking for a Roblox script executor, Lua executor, or script hub on Windows, Wave is organized to cover those needs in one desktop package.
